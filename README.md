1. Clonar proyecto
2. Copiar el ```env.template``` y renombrarlo por ```.env```
3. Ejecutar 
```
npm i
```
4. Levantar imagen de docker
```
docker-compose up -d
```
5. Levantar servidor 
```
npm run start:dev
```
